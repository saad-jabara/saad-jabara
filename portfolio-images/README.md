# Portfolio Images

This folder contains images for the GitHub profile README projects section.

## Required Images

Please add the following images to this folder:

1. **rag-project.jpg** - Screenshot or diagram of your RAG From Scratch project
2. **movie-correlation.jpg** - Visualization from the Movie Correlation Analysis project
3. **web-scraper.jpg** - Screenshot of the Amazon Web Scraper in action
4. **sql-cleaning.jpg** - Screenshot or diagram of SQL data cleaning process
5. **sql-exploration.jpg** - COVID-19 data exploration visualizations
6. **tableau-dashboards.jpg** - Screenshot of your Tableau dashboards

## Image Guidelines

- **Format**: JPG or PNG
- **Recommended size**: 800-1200px width
- **Aspect ratio**: 16:9 or 4:3 works best
- **Quality**: High resolution, clear and readable text
- **Content**: Show key features, visualizations, or outputs from each project

## Adding Images

1. Save your project screenshots with the exact filenames listed above
2. Place them in this `portfolio-images/` folder
3. Commit and push to GitHub

The README.md file already references these images and will display them automatically once added.
